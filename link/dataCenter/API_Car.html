包含车辆数据相关操作的接口

# 通过tuid获取车辆租户ID和用户ID（Local）

功能描述：通过tuid获取少量的车辆信息 包括车辆对应的租户ID和所属的用户ID，此接口有本地缓存

路由 :::: tabs ::: tab Dubbo方式 interface：com.changan.carbond.dubbo.IGlobalUserCarDubboService

method：getSimpleUserCarByTuid ::: ::: tab SpringCloud方式 service name: carbond-base-service

request uri: /api/v1.0/car

request method：GET ::: ::: tab HTTP方式 request url:http://SERVER/api/v1.0/car

request method：GET ::: ::::

输入参数：

参数 必须 类型 含义
tuid 是 String 设备ID

输出参数：

参数 数据类型 说明
statusCode int 状态码 成功 0 失败 其他
data 响应数据 此处为简单的用户车辆信息
message String 消息信息

输出参数中data参数解释：

字段 类型 含义 备注
carId String 车辆ID
tenantId String 租户ID
userId String 用户ID

# 通过车辆ID获取车辆基本信息

功能描述：通过车辆ID获取车辆的基本信息

路由 :::: tabs ::: tab Dubbo方式 interface：com.changan.carbond.dubbo.IGlobalUserCarDubboService

method：getFullCarInfoByCarId ::: ::: tab SpringCloud方式 service name: carbond-base-service

request uri: /api/v1.0/car/getFullCarInfoByCarId

request method：GET ::: ::: tab HTTP方式 request url:http://SERVER/api/v1.0/ca/getFullCarInfoByCarIdr

request method：GET ::: ::::

请求方式：GET

输入参数：

参数 必须 类型 含义
carId 是 String 车辆ID

输出参数：

参数 数据类型 说明
statusCode int 状态码 成功 0 失败 其他
data 响应数据 此处为用户车辆信息
message String 消息信息

输出参数中data参数解释：

字段 类型 含义 备注
carId String 车辆ID
vin String VIN
isCaCar Boolean 是否长安车
isNev Integer isNev 0： 乘用车 1： 新能源 2： 商用车
tenantId String 租户ID
userId String 用户ID
companyId Long 企业ID
carName String 车辆名称
realnameType Integer 0:个人，1：企业，2：展车
carType String 车辆类型
oilType String 汽油类型
brandId String 品牌ID
brandCode String 品牌编码
brandName String 品牌名称
seriesId String 车系ID 使用编码字段进行搜索
seriesCode String 车系编码
seriesName String 车系名称
modelId String 车型ID 使用编码字段进行搜索
modelCode String 车型编码
modelName String 车型名称
confId String 配置ID 使用编码字段进行搜索
confCode String 配置编码
confName String 配置名称
img String 车辆图片路径
engineNo String 发动机号
color String 颜色
purchasedDate Date 购买日期
factoryDate Date 出厂日期
oemCompanyId String 车厂公司ID
capacity String 载客量
plateNumber String 车牌号
plateTime Date 车辆首次上牌时间
pin String 设备控制PIN码
insuranceEndDate Date 保险到期时间
registerTime Date 注册时间
activeTime Date 激活时间
lastCheckTime Date 上次年检时间
insuranceCompany String 保险公司
insurancePhone String 保险公司电话
productTime Date 出厂时间
dealerId String 经销商ID
materialId String 物料ID
tserviceStatus String T服务状态 active-启用 inactive-停用 expire-已到期
tserviceStarttime Date T服务开始时间
tserviceEndtime Date T服务到期时间
lastOnlineTime Date 最后在线时间，从car_condition_json中读取
carAuthCode String 身份实名认证验证码
realnameAuthMobile String 实名认证使用的手机号
realnameAuthIdcardType String 实名认证证件类型
realnameAuthIdcard String 实名认证证件信息
realnameAuthStatus String 实名认证状态
realnameAuthPassTime Date 实名认证通过时间
certifiNo String 合格证书编号
source String 信息来源
deleted Integer 是否删除 1-删除 0-未删除
maintainSpaceMileage Float 剩余保养里程
lastMaintainMileage Float 上次保养里程
totalOdometer Float 总里程
powerPrice Float 电价
issuedate Date 合格证书发放时间
systemCode String 车辆所属机构编码
isPrivacyDrvingHistory Boolean 轨迹是否隐私
