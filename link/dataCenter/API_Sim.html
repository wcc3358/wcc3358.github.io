# 通过Iccid获取Sim卡信息

功能描述：使用iccid获取sim卡的基本信息

路由

:::: tabs ::: tab Dubbo方式 interface：com.changan.carbond.dubbo.ISiSimInfoDubboService

method：getSimInfoByIccid ::: ::: tab SpringCloud方式 service name: carbond-base-service

request uri: /api/v1.0/sim

request method：GET ::: ::: tab HTTP方式 request url:http://SERVER/api/v1.0/sim

request method：GET ::: ::::

输入参数：

参数 必须 类型 含义
iccid 是 String IC卡的ID

输出参数：

参数 数据类型 说明
statusCode int 状态码 成功 0 失败 其他
data 响应数据 此处为sim卡信息
message String 消息信息

输出参数中data参数解释：

monthtodateusage BigDecimal 本月累计流量(MB)
overagelimitreached Boolean 已达到流量限额
