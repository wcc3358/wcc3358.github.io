# 设备API

包含车辆设备及其扩展信息相关操作的接口

# 通过设备ID获取设备基本信息

功能描述：通过设备ID获取设备的基本信息

路由 :::: tabs ::: tab Dubbo方式 interface：com.changan.carbond.dubbo.ICoreTerminalDeviceDubboService

method：getCoreTerminalDeviceByTuid ::: ::: tab SpringCloud方式 service name: carbond-base-service

request uri: /api/v1.0/device

request method：GET ::: ::: tab HTTP方式 request url:http://SERVER/api/v1.0/device

request method：GET ::: ::::

输入参数：

参数 必须 类型 含义
tuid 是 String 设备ID

输出参数：

参数 数据类型 说明
statusCode int 状态码 成功 0 失败 其他
data 响应数据 此处为设备的基本信息
message String 消息信息

输出参数中data参数解释：

字段 类型 含义 备注
tuid String TUID
tenantId String 租户ID
uuid String UUID
carId String 车辆ID
vin String 车辆VIN
providerId String 供应商ID
providerName String 供应商名称 OBD/TBOX以及每个设备对应的厂家
imei String imei
protocolCode String 协议代号
phone String delete
msisdn String 手机号码
carType String 车辆类型
status String 设备状态 active：无效设备inactive 已启用active 已废弃obsolete 运输状态transport
activeTime Date 终端激活时间
authCode String 验证码
deviceName String 设备名称
deviceType String 设备类型
softVersion String 版本号
hardwareVersion String 硬件版本
softwareNum String 软件件号
lastUpgradeCheckVersion String 上次检查版本
hardwareNum String 硬件件号
lastUpgradeCheckTime Date 最近升级检查的时间
deviceSeriesId String 设备型号ID
deviceSeries String 设备型号
deviceSeriesCode String 设备系列编码
dataSource String 信息来源
bindTime Date 车辆与设备绑定时间
productTime Date 出厂时间
discardTime Date 废弃时间
registerTime Date 注册时间
productionNumber String 设备生产序列号
certId String 设备证书id
deleted Integer 是否删除 1-删除 0-未删除
partNumber String 设备件号
createAt Date 创建时间
createBy String 创建者
updateAt Date 更新时间
updateBy String 更新人

# 通过设备ID获取设备扩展信息

功能描述：通过设备ID仅获取设备的扩展信息

路由 :::: tabs ::: tab Dubbo方式 interface：com.changan.carbond.dubbo.ICoreTerminalDeviceExtDubboService

method：getCoreTerminalDeviceExtByTuid ::: ::: tab SpringCloud方式 service name: carbond-base-service

request uri: /api/v1.0/device/ext

request method：GET ::: ::: tab HTTP方式 request url:http://SERVER/api/v1.0/device/ext

request method：GET ::: ::::

输入参数：

参数 必须 类型 含义
tuid 是 String 设备ID

输出参数：

参数 数据类型 说明
statusCode int 状态码 成功 0 失败 其他
data 响应数据 此处为设备的扩展信息
message String 消息信息

输出参数中data参数解释：

字段 类型 含义 备注
tuid String TUID
infoExt String 扩展信息 json字符串格式
createAt Date 创建时间
createBy String 创建者
updateAt Date 更新时间
updateBy String 更新人
