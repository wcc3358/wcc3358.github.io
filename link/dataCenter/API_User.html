# 通过用户ID获取用户信息

功能描述：使用userId获取到用户的基本信息

路由 :::: tabs ::: tab Dubbo方式 interface：com.changan.carbond.dubbo.IGlobalUserCarDubboService

method：getFullUserInfoById ::: ::: tab SpringCloud方式 service name: carbond-base-service

request uri: /api/v1.0/user

request method：GET ::: ::: tab HTTP方式 request url:http://SERVER/api/v1.0/user

request method：GET ::: ::::

输入参数：

参数 必须 类型 含义
userId 是 String 用户ID

输出参数：

参数 数据类型 说明
statusCode int 状态码 成功 0 失败 其他
data 响应数据 此处为用户信息
message String 消息信息
